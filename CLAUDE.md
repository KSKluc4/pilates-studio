# Pilates Studio Management System

App web fullstack para gerenciar um estúdio de pilates.

## Stack

- Next.js (frontend)
- Express.js (backend/API)
- MongoDB (banco de dados)
- JWT para autenticação

## Funcionalidades

- Login seguro com roles (admin e recepcionista)
- Cadastro e listagem de pacientes
- Agendamento de aulas
- Histórico de pagamentos

## Organização

Código limpo, seguro e bem organizado, em pastas separadas para frontend e backend.
